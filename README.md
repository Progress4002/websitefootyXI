# Football Player Stats Website

This is a single-page football analytics website that showcases different players and positions with written analysis and statistics, built as a static HTML page.

## Features

- Sections for key positions (Forward, Midfielder, Defence)
- Player-focused writeups with narrative-style analysis
- Clean layout suitable for desktop viewing
- Simple, framework-free implementation using plain HTML and CSS

## Technologies Used

- HTML5
- CSS3


## Getting Started

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Edge, Safari).

### Running the Website

Open the index.html file in your browser:

- Double-click index.html, or  
- Right-click index.html and choose “Open with” → your browser.
